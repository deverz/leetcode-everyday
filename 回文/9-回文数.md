[9. 回文数](https://leetcode-cn.com/problems/palindrome-number/)

```
给你一个整数 x ，如果 x 是一个回文整数，返回 true ；否则，返回 false 。
回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

例如，121 是回文，而 123 不是。
```

```
思路：
1、判断必然不是回文和必然是回文的特殊情况
    1.1、负数必然都不是回文
    1.2、能被10整除的正数必然都不是回文
    1.3、小于10的正数必然都是回文
    
2、流程分析
根据数学方法取余操作将数字进行反转，不需要全部反转，只需要返回一半数字即可
然后将反转的一半数字和剩下的一半数字进行对比，相等则表示是回文
注意这里存在奇数和偶数的情况判断

比如123321：反转取一半123和剩余的123比较
比如12321：反转取一半123和剩余的12比较，这里比较方式是 123/10 == 12

时间复杂度：O(logn)
```

```go
func isPalindrome(x int) bool {
	// 负数，正数能被10整除 都不是回文数字
	if x < 0 || (x > 0 && x%10 == 0) {
		return false
	}
	// 小于10的数，肯定是了，不再处理了
	if x < 10 {
		return true
	}
	// 反转x的一半，和剩下的比较
	rev := 0 // 倒置变量，初始为0
	// 当rev小于x时，一直进行反转
	for rev < x {
		// 从个位数开始反转数字
		rev = rev*10 + x%10
		x = x / 10
	}
	// x==rev 表示是个偶数，反转的一半和剩下的一半相等，那么是回文
	// x == rev/10 表示是个奇数，反转的一半比剩下的一半在个位上多了一位，去除个位，然后比较
	return x == rev || x == rev/10

}
```
