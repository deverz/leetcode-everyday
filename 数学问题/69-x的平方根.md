[69. x 的平方根](https://leetcode-cn.com/problems/sqrtx/)

```
69. x 的平方根 
给你一个非负整数 x ，计算并返回x的 算术平方根 。
由于返回类型是整数，结果只保留 整数部分 ，小数部分将被 舍去 。
注意：不允许使用任何内置指数函数和算符，例如 pow(x, 0.5) 或者 x ** 0.5 。

示例 1：
输入：x = 4
输出：2

示例 2：
输入：x = 8
输出：2
解释：8 的算术平方根是 2.82842..., 由于返回类型是整数，小数部分将被舍去。
```
```
思路：
二分查找
1、如果a的平方等于x，则a肯定是需要的整数
2、如果a的平方大于x，则a肯定不是需要的整数
3、如果a的平方小于x，则a可能是需要的整数

所以通过二分查找来找a的值，等于就输出，大于就往左边找，小于就往右边找看看还有没有小于的
特殊值为0和1，x=0或者1时，直接返回0和1即可

正常来说，x的开平方最大为x/2，所以二分查找从[1,x/2]开始

二分查找的时间复杂度为O(logN)
```
```go
func mySqrt(x int) int {
	// 如果是0或者1则直接返回
	if x == 0 || x == 1 {
		return x
	}
	// 定义二分查找左右边界
	l, r := 1, x/2
	// 二分查找开始
	for l < r {
		// 获取区间的中间值
		mid := l + (r-l+1)/2
		// 用除法替代 x < mid*mid 防止数字溢出
		if mid > x/mid {
			// 中间值的平方大于x，那么肯定不是要找的数字，需要往左边区间找
			r = mid - 1
		} else {
			l = mid
		}
	}
	return l
}
```
